Tetris engine on Rust
=====================

Deployment
----------

Just clone:

```
$ git clone https://github.com/digitalorder/tetrust.git
$ cd tetrust
```

Run
---

This is a minimum playable game:

```
$ cargo build --release
$ cargo run
or
$ target/release/tetrust
```

Tests
-----

All tests are expected to be green:

```
$ cargo test
```
